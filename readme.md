# personrepo

Installation

1. clone: https://github.com/mokgosi/personrepo.git

2. From projet folder, run

   -- composer update 

3. From projet folder, run the following commands to complete installation

   -- php artisan migrate

   -- php artisan db:seed

   -- php artisan serve 

4. Browse to http://localhost:8000/
